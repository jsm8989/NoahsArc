Welcome to the new website

New team website

FEBRUARY 6, 2021

We are glad to have you here!

This website was produced by the team as a central point for people to find out about us, including other scientists, sponsors and anyone who is interested.

As you can tell it is still getting used to the world, so if something doesn't seem to be working properly, please let us know on the contact form.

Web design is an interesting thing to learn a bit about, but we as a team have barely even scratched the surface. In any case, we hope you will find it useful and that it works in most cases; we decided it was worth the challenge of writing our own code instead of using any website of the many web design platforms out there.

The plan for the website is to complement our social media outreach with blog posts, and to act as a centralised platform for those who may not use various social media platforms. You can find more information about the team on the about page, and we will be updating the pages with scientific information and links to mentions in the press. Stay tuned for more content!

**Technical details**

Using a template from Free Website Templates to get a general structure going, this was written in HTML and CSS with some JavaScript. The website is hosted on GitHub.
